# Smart-Timetable-Assistant-AI-Agent
-> A Streamlit-based academic timetable management application designed to help students organize classes, assignments, and exams.
This project is developed as part of the Smart Timetable Assistant AI Agent
=> Tech Stack 
Frontend / UI: Streamlit
Backend: Python
Data Handling: Pandas
Storage: Session-based (local, no database yet)
✨ Features 
Add timetable events (Event name, date, start time, end time)
Display all events in a tabular timetable view
Simple and clean Streamlit UI
Runs locally on browser
