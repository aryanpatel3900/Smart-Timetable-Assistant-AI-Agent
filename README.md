# Smart-Timetable-Assistant-AI-Agent
📅
Smart Timetable Assistant AI Agent
An AI-powered student schedule manager built with Streamlit + Groq (LLaMA 3).

✨ Features
Add events with conflict detection
Assignment tracker with priority levels
Exam schedule & study planner
Find free time slots
Email reminders
AI assistant for scheduling queries
🛠️ Tech Stack
Frontend
Streamlit
AI Model
Groq / LLaMA 3.3
Language
Python 3.10+
Deployment
Streamlit Cloud
⚙️ Setup
git clone https://github.com/aryanpatel3900/Smart-Timetable-Assistant-AI-Agent.git
pip install -r requirements.txt
streamlit run app.py
🔑 .env File
GROQ_API_KEY=your_groq_key
EMAIL_ADDRESS=youremail@gmail.com
EMAIL_PASSWORD=your_app_password
